# turing-machine

Turing Machine sample program written by Clojure

## Usage
    git clone https://github.com/snufkon/turing-machine
    cd turing-machine
    lein run add.clj "[1 1 1 nil 1 1]"  <-- calculate 3 + 2
    lein run mul.clj "[1 1 1 1 :x 1 1]" <-- calculate 4 * 2
